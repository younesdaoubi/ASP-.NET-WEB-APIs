Bienvenue dans SpaceManagement, une application web dédiée à la gestion des objets célestes ! Pour commencer, téléchargez ou clonez le projet via Visual Studio. Ce projet inclut deux APIs pour le backend (CRUD et AUTH) et une interface frontend en React (JavaScript). Si vous souhaitez profiter de l'interface graphique, lors du premier lancement, exécutez la commande npm install pour installer les modules nécessaires (car ces derniers ne sont pas inclus lors des commits), puis lancez npm start pour démarrer l'application.
